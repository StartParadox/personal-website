Projects

Poo-P

Created at the William Osler KissMyApp Hackathon, this is an android application that would give a stool/urine analysis to the user (health issues, solutions and prevention). The app would give the user a series of questions to answer (colour, shape, symptoms etc.) to determine if there were any potential problems.

PythonGrow

This web application was a finalist at the Koding Global Virtual Hackathon. The aim of this application was to introduce software development to beginners through gamification. The web application developed was based on the classic game of snake. Players would navigate the python to eat letters in the correct order to write out a snippet of code.

PebbleChange

A Pebble Watch application developed at UofTHacks 2015 based on the theme of social benifit. Users are given a list of charities (choosing from local,national or internation) that they can donate their spare change to. They can access more information about the charity through QR codes displayed on the Pebble. The application also showed trending hashtags related to charities on Twitter (such as the ice bucket challenge)

PillowCase

Finalist at WearHacks Toronto 2015, this is an internet of things device/pillow that monitors a users head movement and sound (ex. snoring) and then displays it as a real-time graph on the web for the users to view to identify health risks.
